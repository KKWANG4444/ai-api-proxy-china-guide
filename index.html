📋 快速接入步骤

注册并获取 API Key

访问 www.aifast.club 注册账号，在控制台创建 API Key。支持国内主流支付方式。

配置 Base URL

在任何兼容 OpenAI SDK 的工具中，将 Base URL 设置为 https://www.aifast.club/v1，填入你的 API Key。

选择模型，开始调用

支持 Claude Opus 4.7、GPT 5.5 Pro、DeepSeek V4、Gemini 3、Grok 4.2 等 572 个模型。修改 model 参数即可切换。

集成到你的工具链

Cursor、Windsurf、Dify、LobeChat、NextChat、Cline……一个 Key 通吃所有工具。

🔧 常见工具配置

📝 Cursor

Settings → Models → 添加 Base URL: https://www.aifast.club/v1，填入 API Key，选择模型即可。

📝 Dify

Settings → Model Provider → 添加自定义 API，Base URL: https://www.aifast.club/v1。

📝 LobeChat / NextChat

设置界面 → 自定义 OpenAI API → Base URL 填入 https://www.aifast.club/v1。

📝 Python SDK

from openai import OpenAI
client = OpenAI(
    base_url="https://www.aifast.club/v1",
    api_key="你的API Key"
)
response = client.chat.completions.create(
    model="claude-opus-4-7",
    messages=[{"role": "user", "content": "Hello!"}]
)
